# UI改善まとめ（2026/01/17）

## モバイル表示の改善

### ヘッダー部分の2段構成
- **1段目**: ヒントテキスト（左）+ 検索バー（右寄せ）
- **2段目**: 「解消済除く」チェックボックス + ソートドロップダウン

### ハンバーガーボタンの移動
- 左上 → **左下**に移動
- 半透明（rgba 0.5）+ ぼかし効果
- 親指でアクセスしやすい位置に

### ソートドロップダウン追加（モバイル専用）
- 非表示カラムでもソート可能に
- 選択肢: 報告日↓新/↑古、解消日↓新/↑古、情報源↑/↓
- 「～件表示」セレクトを非表示にしてスペース確保

### その他モバイル調整
- 上部余白削減（60px → 16px）
- ヒントテキスト: 「👆 タップで詳細情報を表示」に短縮
- ヘッダー行間を詰めてコンパクトに

## 共通（デスクトップ・モバイル）

### DataTables設定
- デフォルト表示件数: 20件
- デフォルトソート: **報告日（新しい順）**

### 解消済フィルター
- チェックボックスでON/OFF
- デフォルトでチェック済み（解消済を除外）
- 解消日が未来の場合は表示
